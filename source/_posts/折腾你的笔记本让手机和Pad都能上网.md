### 寝室热点

首先笔记本都有两个网卡，一个无线的，一个有线的。

当然啦，一个网卡只能做一件事。如果在家里或者宿舍里，只有一个账号能上网。身边又没有路由器或者懒得捣鼓了就按照如下方式做吧

原理通俗的讲是这样的。首先笔记本是插入网线上网的，让后笔记本无线网卡的流量从有线网口走出去，这样有线的无线的都能上网，效果就是：你的笔记本成为一个无线热点了，而且只有知道密码的才能用。

步骤:
- 打开CMD(只限于windows)。输入如下命令
在cmd下输入

``` dat
netsh wlan set hostednetwork mode=allow ssid=open key=12345678
```
这样就设置好了，网络名称是open,密码是12345678
然后开启这个热点就行了。
输入

```data
netsh wlan start hostednetwork
```

如果显示“已启动承载网络”就说明成功了。拿你的手机连上去输入密码12345678就可以上网啦。不过别忘了笔记本先要先上网才行。
图片上的是启动失败了的显示，因为有的笔记本的无线网卡不支持这个功能。所以可能会失败的。失败的情况还是比较少的。

enjoy!
